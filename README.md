# CRGuild

길드 창설·관리, 신호기(Beacon) 기반 영토 시스템, 전쟁·항복 메카닉을 제공하는 Paper 플러그인입니다.

## 지원 버전

| 항목 | 버전 |
|------|------|
| Minecraft | 1.20.4 |
| Paper API | 1.20.4-R0.1-SNAPSHOT |
| Java | 21 |
| 프레임워크 | CRFramework 1.0.0 |

## 기능

- **길드 창설** — 신호기(Beacon) 설치 시 자동 영토 등록 및 길드 창설
- **직책 시스템** — 길드장 / 부길드장 / 일반 멤버 3단계 권한 구조
- **국고(Treasury)** — 길드 공용 자금 입출금, 레벨업·세금·항복 비용 자동 처리
- **레벨업** — Lv.1~5, 레벨에 따라 최대 인원 확장
- **주간 세금** — 미납 3회 누적 시 길드 자동 해산
- **길드 채팅** — `/길드 채팅` 토글로 길드원 전용 채팅 채널 사용
- **성 이동 GUI** — 등록된 성으로 카운트다운 텔레포트 (전쟁 중 10초, 평시 3초)
- **전쟁 시스템** — 선포권 아이템 소모 → 5분 준비 단계 → 전쟁 돌입
- **항복** — 누적 패배에 따라 증가하는 비용 구조 (수비 길드만 가능)
- **성 파괴 & 길드 멸망** — 전쟁 중 폭탄 아이템으로 적 영토 파괴, 모든 성 파괴 시 멸망
- **다중 저장소** — YAML / MySQL 선택 가능 (HikariCP 커넥션 풀)
- **메시지 커스터마이즈** — config.yml 에서 모든 메시지를 `&` 색상 코드와 함께 편집 가능

## 의존성

| 플러그인 | 필수 여부 |
|----------|-----------|
| [CRFramework](https://github.com/zlero7/CRFramework) | **필수** |
| [Vault](https://www.spigotmc.org/resources/vault.34315/) | **필수** (경제 연동) |
| [WorldEdit](https://enginehub.org/worldedit) | 선택 (영토 보호 기능 사용 시) |

## 설치

1. `CRFramework-*.jar`, `Vault.jar`, `CRGuild-*.jar` 를 `plugins/` 폴더에 복사
2. 서버 시작 후 생성된 `plugins/CRGuild/config.yml` 편집
3. 서버 재시작

## 설정

```yaml
war:
  min-online: 1                      # 전쟁 시작에 필요한 길드원 최소 온라인 수
  surrender-base-cost: 1000000       # 항복 기본 비용 (1회차, 단위: 원)
  surrender-cost-increase: 1000000   # 항복 회차당 추가 비용

storage:
  type: yaml      # yaml | mysql  (yaml = 파일 저장, mysql = DB 저장)
  host: localhost
  port: 3306
  database: crguild
  username: root
  password: ""    # 실제 비밀번호는 서버 config.yml 에만 기재

messages:         # 모든 메시지를 & 색상 코드로 자유롭게 편집 가능
  common:
    no-guild: "&c[!] 소속된 길드가 없습니다."
  war:
    declare:
      broadcast:
        - "&8[ &4⚔ 전쟁 선포 &8] &f%attack_guild% &7vs &f%defend_guild%"
  # ... 이하 config.yml 참고
```

## 커맨드

### 일반 커맨드

| 커맨드 | 설명 |
|--------|------|
| `/길드` | 내 길드 정보 조회 |
| `/길드 선포 [이름]` | 길드 창설 선포 (100만원 차감) |
| `/길드 정보 [길드명]` | 길드 정보 조회 |
| `/길드 초대 [플레이어]` | 멤버 초대 (부길드장↑) |
| `/길드 수락 / 거절` | 초대 수락·거절 |
| `/길드 추방 [플레이어]` | 멤버 추방 (부길드장↑) |
| `/길드 탈퇴` | 길드 탈퇴 (24시간 재가입 쿨타임) |
| `/길드 해산` | 길드 해산 (길드장) |
| `/길드 국고 입금/출금 [금액]` | 국고 입출금 |
| `/길드 레벨업` | 국고 차감으로 레벨업 (길드장) |
| `/길드 공지 [내용]` | 공지 설정 (부길드장↑) |
| `/길드 임명/해임 [플레이어]` | 부길드장 관리 (길드장) |
| `/길드 위임 [플레이어]` | 길드장 위임 (길드장) |
| `/길드 목록` | 전체 길드 목록 |
| `/길드 이동` | 성 이동 GUI 열기 |
| `/길드 채팅` | 길드 채팅 ON/OFF 토글 |

| 커맨드 | 설명 |
|--------|------|
| `/전쟁 선포 [길드명]` | 전쟁 선포 (선포권 아이템 소모, 길드장) |
| `/전쟁 항복` | 항복 선언 (수비 길드·길드장, 2단계 확인) |
| `/전쟁 현황` | 진행 중인 전쟁 목록 |

### 관리자 커맨드

**권한**: `crguild.admin` (기본값: OP)

| 커맨드 | 설명 |
|--------|------|
| `/길드관리 해산 [길드명]` | 길드 강제 해산 |
| `/길드관리 정보 [길드명]` | 길드 상세 정보 조회 |
| `/길드관리 국고 [길드명] [금액]` | 국고 강제 입금 |
| `/길드관리 레벨 [길드명] [1-5]` | 레벨 강제 설정 |
| `/길드관리 추방 [플레이어]` | 강제 추방 |
| `/길드관리 목록` | 전체 길드 상세 목록 |
| `/전쟁관리 시작 [길드A] [길드B]` | 준비 없이 즉시 전쟁 시작 |
| `/전쟁관리 무승부 [길드A] [길드B]` | 전쟁 무승부 강제 종료 |
| `/전쟁관리 현황` | 전쟁 목록 조회 |
| `/전쟁관리 선포권 [플레이어] [수량]` | 전쟁 선포권 지급 |

## 빌드

```bash
./gradlew shadowJar
```

결과물: `build/libs/CRGuild-1.0.0.jar`

> `jar` 태스크가 아닌 **반드시 `shadowJar`** 를 사용해야 합니다.  
> HikariCP가 함께 번들링됩니다.
